UKP Instant Messenger

UKP is a open-source instant messenger, made in the year 2022. It contains many useful features, such as sending photos, text files and actually, any file. You can also view photos/images in-messenger (no need to click on it) the user who's talking to you sent you. And also, it contains slash commands. For example, you can type commands like:
/clear (Clears the chat, the user who you're talking to won't see it.)
/gif {type} (Sends any GIF file, ex: /gif dog)
.. And much much more! So, what are you waiting for? Join UKP now!
